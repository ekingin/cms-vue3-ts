# imp-cms-vue3-ts

这是一个后台管理系统，基于 vue3、vue router、pinia、ts、element plus、axios 等。

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
